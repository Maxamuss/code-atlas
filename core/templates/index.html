What is Code Atlas?

Code Atlas was created in spite of wasting so much time searching for trivial things I knew I had done before. I often found myself looking through previous projects or tracking down articles for that code snippet I knew existed. I thought there had to be a better way...
  1. 1

    Create a Note

    Notes are for anything that was hard to find and even harder to remember. If you think it might be useful in the future, add it to a note.

  2. 2

    Search your Notes

    You're stuck. You can't remember how you solved that tricky problem you spent hours on a few months ago. Oh wait! By searching your Notes, you can easily find what you did last time!

  3. 3

    Edit and Improve

    Edit your Notes at any time, add references as well as get direct results from the Django documentation.

Why use Code Atlas?

It can be tricky to keep your notes in one place. Add in bookmarks and as soon as you lose discipline, you're in the same situation of scattered notes and references all over again. With Code Atlas, everything is kept together. Made for making your life easier, features include:

Markdown Editing
Tagging
Url References
Quick Search
Documentation Search

But is Code Atlas for me?

No matter your programming experience, you can benefit from using Code Atlas:
Starting out takes time. Learning something new never comes easy. Anytime you solve a tricky problem, summerise the solution and add it to a Note. Trust me, your future self will thank you.
Beginner
There's only so much you can keep in your head. By using Code Atlas, you can free up that brain space for new knowledge, improving your projects and personal development.
Hobbyist
If you don't have countless bookmarks and text documents spewed all over the place, your a better person than me. Consolidate all these in one place and easily create, edit and search these Notes.
Experienced Developer
